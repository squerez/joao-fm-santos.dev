## Installing and running locally 

Clone the project

```bash
git clone https://github.com/squerez/joao-fm-santos.dev.git
```

Go to the project directory

```bash
cd joao-fm-santos.dev
```

Install dependencies

```bash
npm install
```

Start the server

```bash
npm run dev
```
